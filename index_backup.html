BreathBubbles

Breath Bubbles

一个具有沉浸式 呼吸泡泡 动效的静态站点示例。通过柔和的节奏、轻盈的粒子与现代 UI 设计，帮助用户进入专注与放松的流动状态。

立即体验 Github

核心特性

呼吸节奏

基于正弦曲线的缓慢膨胀与收缩，营造“吸气 / 呼气”韵律，让视觉与心率同步放缓。

交互驱动

鼠标 / 触控轻推泡泡产生避让与联动连线，营造轻量的互动反馈。

性能友好

原生 Canvas + requestAnimationFrame，限制粒子数与动态更新范围，保持流畅。

现代美学

暗色渐变、低饱和霓虹点缀、玻璃拟态卡片，与可调式主题变量。

模块化结构

CSS 变量 + 组件化布局，便于拓展为产品落地页或冥想工具。

Hugo 集成

使用 Hugo Pipes 进行资源 Fingerprint 与压缩，适合部署与缓存。
